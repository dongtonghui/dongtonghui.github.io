CHAPTER 03
如何安装
从零开始，5分钟拥有你的AI Agent
🖥️ 本地安装
$ pip install hermes-agent # 安装核心包 $ hermes setup # CLI初始化 $ hermes webui start # 启动WebUI $ ✓ WebUI running on 0.0.0.0:8082 $ hermes skills list # 查看可用技能
CLI终端 + WebUI图形界面，双模式操作
☁️ 云端安装
$ ssh root@你的服务器 # 登录阿里云 $ apt install -y docker.io # 安装Docker $ docker pull hermesai/hermes-webui # 拉取镜像 $ docker run -d -p 8082:8082 hermesai/hermes-webui $ ✓ Container started · 云端就绪
