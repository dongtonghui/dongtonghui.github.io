安装 Hermes · WebUI

在 CLI 基础上，启动 Hermes WebUI 即可获得图形化操作界面，一站式管理对话、模型、工具和技能。

01
🚀

启动服务

运行以下命令启动 WebUI 服务，默认监听 9119 端口。

hermes dashboard --port 9119
→
02
🔗

浏览器访问

打开浏览器，访问本地地址即可进入 WebUI 控制台。

http://localhost:9119
→
03
⚙️

个性配置

在 WebUI 中可视化配置模型提供商、启用工具、安装和管理技能。
