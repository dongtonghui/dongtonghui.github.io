安装 Hermes · CLI 模式

一行命令，即刻上手。Hermes CLI 是体验 Hermes 核心能力最快速的方式，支持交互对话与脚本集成。

01
一键安装 
$ curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash

自动检测系统环境，安装 Hermes 到本地，无需手动配置依赖。

→
02
交互对话 
$ hermes

启动交互式终端，直接开始对话。Hermes 自动加载工具和技能。

→
03
单次查询 
$ hermes chat -q "分析这份PRD文档"

无需进入交互模式，直接传入问题参数，快速获取回答。

🔄
切换模型

运行 hermes model 实时切换 AI 底层模型

📋
管道集成

支持 echo "prompt" | hermes 管道模式

⚙️
技能管理

运行 hermes skill 管理已安装的技能

🖥 支持平台：
🐧 Linux 🍎 macOS 🪟 WSL2
